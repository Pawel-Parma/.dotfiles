# Dotfiles
My dotfiles


# Installation
## 1. Install requirements:
- [stow](https://www.gnu.org/savannah-checkouts/gnu/stow/stow.html)

## 2. Clone repository 
```sh
git clone --depth 1 --recurse-submodules --shallow-submodules https://github.com/Pawel-Parma/.dotfiles.git
```

## 3. Install 
```sh
cd .dotfiles
stow -t ~ .
```
